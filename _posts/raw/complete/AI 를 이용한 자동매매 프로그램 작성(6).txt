## 현재 작업 브랜치

- 브랜치명: feature/build

- 작업 내용: 대시보드 차트 기능 구현

- 시작일: 2026-05-30

​

## 작업 시작 전 체크리스트

1. main 브랜치로 전환 후 pull

2. feature/build 브랜치로 전환

3. main 최신 내용 반영 (rebase)

4. 지난 작업은 build-docs/work-log.md 을 확인

5. 이어서 해야 할 내용은 build-docs/next-tasks.md 확인

​

## 작업 종료 절차

1. 변경사항 전체 확인 (git status)

2. 작업 내용 커밋 (커밋 메시지는 작업 내용 간단히)

3. feature/build 브랜치에 push

4. build-docs/work-log.md 에 오늘 작업 내용 기록

5. 다음 작업 시 이어할 내용을 build-docs/next-tasks.md 에 추가해줘
