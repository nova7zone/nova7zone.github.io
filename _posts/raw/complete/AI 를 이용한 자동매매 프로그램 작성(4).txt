Claude Code 를 이용하여 프로그램 수행

Claude Code 에서 저장소를 클라우드로 설정하고 새로 가입한 Github와 연동 - 안됨

일단 로컬에 프로젝트 폴더를 만들고 프롬프트 파일 저장

로컬로 지정을 하니 git for windows 가 필요하다고 해서 설치 후 claude desktop 재시작

Claude code pro에 가입이되어 있는데 Claude code 실행시 벌도 과금이 있는지 확인 필요.

claude code 에 어떻게 작업을 시킬지 어려워서 Gemini 에게 질의

답변: 

----------------

"현재 폴더에 있는 KIS_TRADING_PROMPT_V4_EN.txt 파일을 꼼꼼히 읽고, 명세서에 맞춰 프로젝트 구축을 시작해 줘.

​

특히 파일 마지막의 SECTION 19 [AI 에이전트를 위한 필수 지시사항]을 엄격하게 지켜야 해. 절대 한 번에 모든 코드를 짜지 말고, 가장 먼저 1단계(핵심 프로젝트 구조 생성, requirements.txt, .gitignore, 기본 설정 파일 작성)만 수행해 줘.

​

1단계 작성이 끝나면 터미널에서 git add . 와 git commit -m "chore: initial project setup"을 실행해서 작업 상태를 저장하고, 다음 단계(API 통신 로직 등)로 넘어갈지 나에게 물어봐. 답변은 한글로 해줘"

​

​

💡 진행 팁:

​

이렇게 지시를 내리면 Claude Code가 폴더 구조를 예쁘게 만들고 기본 파일들을 생성한 뒤, 커밋까지 스스로 마칠 거예요. 에이전트가 멈추고 다음 작업을 물어보면, "완벽해! 이제 2단계인 API 클라이언트 부분(kis_api.py, token_manager.py)을 만들어줘" 하는 식으로 대화하듯 이어나가시면 됩니다.

​

혹시 로컬에서 터미널을 열고 Git을 초기화하는 과정에서 조금 헷갈리거나 도움이 필요한 부분이 있으신가요?

​

----------------
1. Claude Code와 Github의 연동을 하였음

2. 여러 컴과 Github의 동기화를 위한 배치파일 생성 완료

3. 추후 새로운 컴퓨터에서 설정 배치파일을 실행 하고 시작 배치 파일을 실행하면 연속 작업 가능

4. 새로운 컴퓨터에서 설정하는 방법

----------------------------------------------------------------

1. setup-new-pc.bat 실행         → 도구 설치 (winget)

2. 터미널 완전히 닫고 새로 열기  → PATH 적용

3. setup-new-pc-step2.bat 실행  → git 설정, GitHub 로그인, 클론, venv

----------------------------------------------------------------

5. 배치 파일을 실행시키는 방법
